# Frontend Module

This folder contains the React + Vite client for the College Student Management System.

For the full project overview, features, setup instructions, and recruiter-facing summary, see the repository root README.

## Frontend Stack

- React 19
- Vite 6
- React Router
- Axios
- React Hot Toast

## Run Frontend Locally

```bash
npm install
npm run dev
```

The frontend expects the backend API to be available at `http://localhost:5000/api`.
